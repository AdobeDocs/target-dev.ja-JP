# [!DNL Dynamic Tag Manager] （DTM）を使用して[!DNL Target]を実装

[!DNL Adobe Dynamic Tag Manager]はレガシー製品であり、サポートされなくなりました。 最適な実装エクスペリエンスを得るために、[!DNL Adobe]は[!DNL Adobe Experience Platform]でタグを使用することをお勧めします。

[!DNL Adobe Experience Platform] のタグは、[!DNL Adobe] の次世代タグ管理機能です。 タグは、適切な顧客体験の実現に必要な分析、マーケティングおよび広告タグをデプロイおよび管理するためのシンプルな手段を提供します。

詳しくは、[実装 [!DNL Target] using [!DNL Adobe Platform]](/help/dev/implement/client-side/atjs/how-to-deployatjs/implement-target-using-adobe-launch.md)を参照してください。
