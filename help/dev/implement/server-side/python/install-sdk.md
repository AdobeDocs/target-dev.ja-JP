# Python SDKのインストール

Python SDKは[pypi](https://pypi.org/project/target-python-sdk)によって配布されています。 開始するには、pip経由でインストールして依存関係として追加します。

## Install

```python {line-numbers="true"}
pip install target-python-sdk
```

オープンソースのコードは[https://github.com/adobe/target-python-sdk](https://github.com/adobe/target-python-sdk)にあります
