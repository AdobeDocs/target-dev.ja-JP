# Java SDKの概要

このリファレンスガイドでは、[!DNL Adobe Target]のお客様がJava SDKをインストール、初期化、使用する方法について説明します。

## 前提条件

* Java 8+
* MavenまたはGradle

## 導入

Java SDKの使用を開始する方法については、[[!DNL Target] SDK入門ガイド ](../sdk-guides/getting-started/getting-started.md)に移動してください。
